table태그 : 표를 만들때 사용하는 태그. 과거에는 테이블태그로 레이아웃도 구성했으나, 현대에는 div태그를 사용하므로 사용빈도가 줄어듬.

tr : 표 내부의 행 (table row)

th : 행 내부의 제목 셀 (table header)

td : 행 내부의 일반 셀 (table data)

colspan 속성 : 셀의 너비 지정 (가로 병합)

rowspan 속성 : 셀의 높이 지정 (세로 병합)
