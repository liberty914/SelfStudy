video태그 : 동영상파일을 재생할수 있게 해주는 태그

src속성 : 파일의 경로 지정

poster속성 : 동영상이 준비중일때, 이미지 파일 경로 지정

preload속성 : 동영상 재생 전에 모두 불러올지 지정

autoplay속성 : 동영상을 자동 재생할지 지정

loop속성 : 동영상을 반복할지 지정

controls속성 : 동영상 재생도구를 출력할지 지정

width속성 : 동영상의 너비를 지정

height속성 : 동영상의 높이를 지정

source태그 : 크로스브라우징 문제를 해결해주는 태그

src속성 : 파일의 경로 지정
