form태그 : 입력양식을 Server로 전송하도록 지원하는 태그

action속성 : 입력 데이터의 전달 위치를 지정 ("/app/user/addUser", "/addUser.do")

method속성 : 입력 데이터의 전달 방식을 선택 (GET, POST ==> 디폴트는 GET방식)

★아래의 태그들은 HTML4부터 지원하던 태그와 속성임.











input태그 : 사용자로부터 데이터를 입력받기위한 태그

type="text" : 글자 입력양식

type="password" : 비밀번호 입력양식

type="hidden" : 숨겨진정보

type="image" : 이미지형태 생성

type="checkbox" : 체크박스

type="radio" : 라디오버튼

type="file" : 파일입력양식

type="reset" : 초기화버튼

type="submit" : 제출(전송)버튼

type="button" : 버튼
